
# AI Mental-Health Journaling Assistant

**Tagline:** Helping you reflect on your emotions, one journal entry at a time.

## Overview
This project is an AI-powered journaling assistant built using the Airia platform.  
Users can write daily journal entries, the system detects emotional patterns, and provides supportive, non-clinical responses.

## Features
- Emotion Detection Agent
- Supportive Response Agent
- Chat-based user interface (Airia Chat Widget)
- Manual evaluation and safety-focused design

## Tech Stack
- Platform: Airia (No-Code)
- Interface: Chat Widget
- Approach: Agent-based workflow

## Current Status
- Core agents created and refined  
- Interface integrated  
- Safety considerations documented  
- Live execution paused due to billing limits (manual evaluation used)


## Future Improvements
- Weekly emotion summaries
- Emotional memory
- Visual dashboard
- Live testing once executions are enabled
